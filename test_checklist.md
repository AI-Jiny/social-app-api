# 테스트 체크리스트

## 인증
- [ ] 정상 회원가입이 가능하다
- [ ] 이메일 중복 시 409를 반환한다
- [ ] username 중복 시 409를 반환한다
- [ ] 로그인 성공 시 access token을 반환한다
- [ ] 로그인 실패 시 401을 반환한다
- [ ] 인증 없이 /me 호출 시 실패한다

## 게시물
- [ ] 로그인 사용자는 게시물을 생성할 수 있다
- [ ] image_url 없이 게시물 생성 시 400을 반환한다
- [ ] 게시물 단건 조회가 가능하다
- [ ] 작성자만 게시물을 삭제할 수 있다
- [ ] 다른 사용자가 삭제 시 403을 반환한다

## 팔로우
- [ ] 로그인 사용자는 다른 사용자를 팔로우할 수 있다
- [ ] 자기 자신 팔로우 시 400을 반환한다
- [ ] 이미 팔로우 중이면 409를 반환한다
- [ ] 언팔로우가 가능하다

## 피드
- [ ] 내가 팔로우한 사용자의 게시물만 조회된다
- [ ] 최신순 정렬이 적용된다
- [ ] limit이 정상 동작한다
- [ ] cursor 기반 다음 페이지 조회가 가능하다

## 좋아요
- [ ] 좋아요 등록이 가능하다
- [ ] 중복 좋아요 시 409를 반환한다
- [ ] 좋아요 취소가 가능하다

## 댓글
- [ ] 댓글 작성이 가능하다
- [ ] 빈 댓글 작성 시 400을 반환한다
- [ ] 게시물 댓글 목록 조회가 가능하다
