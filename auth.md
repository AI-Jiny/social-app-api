# 인증 / 권한 정책

## 인증 방식
- 이메일 + 비밀번호 로그인
- 로그인 성공 시 access token 발급
- 인증이 필요한 API는 Authorization 헤더 사용

## 권한 정책

| 기능 | 비로그인 | 로그인 사용자 | 작성자 |
|---|---|---|---|
| 회원가입 | O | O | O |
| 로그인 | O | O | O |
| 프로필 조회 | O | O | O |
| 게시물 조회 | O | O | O |
| 게시물 생성 | X | O | O |
| 게시물 삭제 | X | X | O |
| 댓글 조회 | O | O | O |
| 댓글 작성 | X | O | O |
| 좋아요 | X | O | O |
| 팔로우 | X | O | O |

## 추가 규칙
- 자기 자신은 팔로우할 수 없음
- 게시물 삭제는 작성자만 가능
- 좋아요는 동일 사용자 기준 1회만 가능
- 댓글은 로그인 사용자만 작성 가능
