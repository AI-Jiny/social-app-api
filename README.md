# Social App API

인스타그램을 클론코딩하여 사진 기반 SNS의 핵심 기능을 구현하는 백엔드 프로젝트입니다.  
사용자는 회원가입 및 로그인 후 게시물을 업로드하고, 다른 사용자를 팔로우하며, 피드에서 게시물을 조회할 수 있습니다.

## 목표
- 인증 기반 사용자 기능 구현
- 게시물, 좋아요, 댓글, 팔로우 기능 제공
- 팔로우 관계를 기반으로 한 피드 조회 API 구현

## MVP 범위

### 포함
- 회원가입
- 로그인
- 내 정보 조회
- 게시물 업로드
- 게시물 단건 조회
- 게시물 삭제
- 팔로우 / 언팔로우
- 피드 조회
- 좋아요 / 좋아요 취소
- 댓글 작성 / 댓글 조회
- 알림

### 제외
- 스토리
- DM
- 해시태그 검색
- 추천 알고리즘
- 비공개 계정
- 대댓글
- 관리자 기능

## 핵심 사용자 흐름
1. 사용자는 회원가입 후 로그인한다.
2. 로그인한 사용자는 게시물을 업로드할 수 있다.
3. 사용자는 다른 사용자를 팔로우할 수 있다.
4. 사용자는 팔로우한 사용자의 게시물을 피드에서 조회할 수 있다.
5. 사용자는 게시물에 좋아요와 댓글을 남길 수 있다.
6. 사용자는 자신의 게시물에 좋아요와 댓글이 생길 경우 알림을 받을 수 있다.
